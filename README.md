# Help Michael Go Home

A 3-level maze game built using Python and Pygame.

## Features
- Player-controlled character (Michael)
- 3 levels with increasing difficulty
- Ghost enemy that chases the player
- Moving obstacles (rock and tree)
- Gold coin scoring system
- Restart option using "R" key
- Ghost warning system (red screen + direction alert)

## Objective
Help Michael collect coins, avoid the ghost, and reach the goal through all 3 levels.

## Built With
- Python
- Pygame

## How to Run
1. Install pygame:

   pip install pygame

3. Run the game:

   python first_pygame.py


## Controls
- Arrow keys → Move Michael  
- R → Restart game
